OPINO - OPINion ONtology: Description back to ToC

OPINO is a content Ontology Design Pattern for representing scholarly interpretive opinions. It extends the Description and Situation (DnS) ODP by introducing epistemic status gradation, inter-opinion relations, and a provenance layer via PROV-O and HiCO. It is informed by the Digital Hermeneutics model (Daquino et al., 2021) and the SEBI ontology (https://valentinapasqual.github.io/sebi/).
