### OPINO: An Ontology for Representing Opinions as Knowledge Graphs - Working Draft

This repository contains the OPINO (OPINion ONtology) ontology, a formal model for representing opinions as Knowledge Graphs. OPINO is grounded in the Description and Situation (DnS) Ontology Design Pattern and the DOLCE+DnS Ultralite foundational ontology (DUL). The ontology is currently in its early stages of development and is intended to serve as a domain-agnostic interface for representing opinions across heterogeneous contexts.
